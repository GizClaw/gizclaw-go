# Server Business RPC

- A registered peer can manage its own pets through server-side RPC.
- A registered peer can claim prompt-driven rewards into its wallet.
- Wallet transactions remain queryable as an audit trail.
- Pet adoption requires `pet_species.use` for the selected species.
- Badge rewards require `badge.use` for the granted badge.
- A peer cannot access another peer's pets, rewards, wallet, or wallet transactions.
